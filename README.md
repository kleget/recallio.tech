# Recallio

Recallio - сервис для изучения иностранных слов по тематическим сферам (биология, IT, математика и т.д.). Он помогает учить слова маленькими ежедневными сессиями и видеть реальный прогресс на графике.

Сайт: https://recallio.tech

## Зачем нужен

- Учить лексику под конкретную область (например, термины для учебы/работы).
- Держать темп без перегруза: немного каждый день, но стабильно.
- Видеть рост словаря и мотивацию через статистику.

## Что есть в сервисе

### Обучение

- **Учить**: новая порция слов на сегодня. Сначала карточки, потом небольшой "перерывной" текст, затем тест на вспоминание.
- **Повторять**: слова, которым пришло время повторения. Сначала тест, затем доучивание забытых.
- Проверка ответов "по смыслу": допускаются небольшие опечатки/варианты. Если у слова несколько переводов - достаточно написать один любой правильный.

### Сферы (корпуса) слов

- Слова сгруппированы по сферам/направлениям.
- Можно выбрать несколько сфер и задать лимиты (сколько слов брать из каждой).
- Есть просмотр слов внутри сферы (чтобы понимать, что именно будешь учить).

### Стартовая настройка

- Язык интерфейса: RU / EN.
- Направление обучения (какой язык знаешь и какой учишь).
- Выбор сфер и лимитов.
- Импорт "уже известных слов" (чтобы сервис учитывал их как выученные/известные).

### Главная страница и статистика

- Показ "учить сегодня" и "повторять сегодня".
- График дисциплины: как растет словарь по дням, с переключением периода.
- Блок коротких советов по эффективному обучению.

### Профиль и настройки

- Профиль пользователя и базовые настройки интерфейса (тема, язык).
- Публичный профиль для социального раздела.

### Сообщество

- Друзья и заявки.
- Общий чат.
- Лента активности (если включено).

### Репорты и админка

- Пользователи могут сообщать о проблемных словах/переводах и артефактах.
- Админ может обрабатывать репорты и править/удалять слова и переводы прямо в админке.
- Аудит действий и ошибок (для диагностики и контроля).

### Уведомления и фоновые задачи

- Напоминания о повторениях (email/telegram; некоторые функции могут быть в разработке).
- Тех-настройки: каналы, время напоминаний, очередь уведомлений (outbox), список фоновых задач.

## Как пользоваться (для пользователя)

1) Открой сайт и зарегистрируйся / войди в аккаунт.
2) Пройди стартовую настройку: выбери направление обучения и сферы.
3) На главной смотри, сколько слов "учить сегодня" и "повторять сегодня".
4) Проходи короткие сессии каждый день - прогресс будет виден на графике.
